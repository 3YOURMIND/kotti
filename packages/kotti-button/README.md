## Usage

### Attributes

| Attribute | Description             | Type      | Accepted values                  | Default |
| :-------- | :---------------------- | :-------- | :------------------------------- | :------ |
| `type`    | button types            | `String`  | primary / second / text / danger | —       |
| `icon`    | button icons            | `String`  | icon name in `yoco`              | —       |
| `loading` | show loading animiation | `Boolean` | —                                | `false` |
