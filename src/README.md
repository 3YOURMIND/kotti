# Kotti-UI

[![npm version][npm-version-badge]][npm-package]

## Install Kotti-UI

```bash
yarn add @3yourmind/kotti-ui @3yourmind/kotti-style
```

## Import Kotti-UI

**Fully Import**

```js
// in main.js
import Vue from 'vue'
import KottiUI from '@3yourmind/kotti-ui'
import '@3yourmind/kotti-style'

Vue.use(KottiUI)
```
