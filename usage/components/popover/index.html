Use popovers to provide extra information or actions. Compared to tooltip, popovers can carry more information. KtPopover supports escaping the CSS Stacking Context

Methods



Scoped Slot

Besides the default slot, KtPopover exposes content slot, which exposes a close function on the slot-scope, which, when called, closes the popper.

Save your message

		<KtPopover trigger="hover">
			<KtButton label="Close with Cancel Button" />
			<template #content="{ close }">
				<p>Save your message</p>
				<KtButton label="Cancel" type="text" @click="close" />
				<KtButton label="Save" type="primary" />
			</template>
		</KtPopover>
